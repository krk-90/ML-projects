# Iris Classification using Logistic Regression

## 📌 Project Overview
This project demonstrates how to classify iris flowers into three species 
(*Setosa*, *Versicolor*, *Virginica*) using **Logistic Regression**.  
The Iris dataset is a classic dataset in machine learning, widely used for 
classification tasks and model evaluation.

---

## 📂 Dataset
- **Source**: The Iris dataset from UCI Machine Learning Repository or `sklearn.datasets`.
- **Features**:
  - Sepal length
  - Sepal width
  - Petal length
  - Petal width
- **Target**: Species of iris flower (Setosa, Versicolor, Virginica)
